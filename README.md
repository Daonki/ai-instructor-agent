# AI Instructor Agent
> PPT 기반 발표 스크립트 자동 생성 및 흐름 제어를 수행하는 LLM 기반 AI Agent 프로젝트

📌 프로젝트 소개
PPT 발표 준비 과정에서 스크립트 작성에 소요되는 시간을 줄이기 위해,  
PPT 파일을 입력하면 슬라이드 내용을 분석해 발표 스크립트를 자동 생성하고  
AI Agent 기반으로 발표 흐름을 지원하는 프로젝트입니다.

KT AIVLE School 팀 미니프로젝트로 진행되었습니다.

---

🛠 사용 기술
- Python
- LLM (OpenAI API)
- Prompt Engineering
- Jupyter Notebook

---

⚙️ 주요 기능
- PPT 슬라이드 내용 분석
- 슬라이드 간 맥락을 고려해 이전·다음 내용을 반영하며 스크립트를 생성
- 발표 흐름에 맞는 스크립트 고도화

---

👩‍💻 담당 역할
- LLM 기반 발표 스크립트 생성 로직 설계
- 프롬프트 구조 설계 및 개선
- 전체 파이프라인 구현

---

📈 결과
- 발표 스크립트 작성 시간 단축
- 슬라이드 흐름에 맞는 일관된 발표 스크립트 생성

---

📂 프로젝트 구성
- `Step1_AI_강의영상_제작.ipynb` : 기본 발표 스크립트 생성
- `Step2_AI_강의영상_고도화.ipynb` : 스크립트 품질 개선
