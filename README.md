# AI Instructor Agent

📌 프로젝트 소개
PPT 파일을 입력하면 슬라이드 내용을 분석하여  
발표 스크립트를 자동 생성하고,  
AI Agent 기반으로 발표 흐름을 지원하는 프로젝트입니다.

KT AIVLE School 팀 미니프로젝트로 진행되었습니다.

---

🛠 사용 기술
- Python
- LLM (OpenAI API)
- Prompt Engineering
- Jupyter Notebook

---

⚙️ 주요 기능
- PPT 슬라이드 내용 분석
- 슬라이드별 발표 스크립트 자동 생성
- 발표 흐름에 맞는 스크립트 고도화

---

👩‍💻 담당 역할
- LLM 기반 발표 스크립트 생성 로직 설계
- 프롬프트 구조 설계 및 개선
- 전체 파이프라인 구현

---

📂 프로젝트 구성
- `Step1_AI_강의영상_제작.ipynb` : 기본 발표 스크립트 생성
- `Step2_AI_강의영상_고도화.ipynb` : 스크립트 품질 개선
